# GRE 核心词汇 第 1 组

## arbitary

| 字段 | 内容 |
|------|------|
| 单词 | arbitrary |
| 释义 | 武断的，任意的；专横的 |
| 音标 | /ˈɑːrbɪtreri/ |
| 例证 | The judge's ruling seemed **arbitrary** and unfair. |
| 词根 | arbit（判断）+ ary（形容词后缀） |
| 拆分 | ar-bi-tra-ry |

## corroborate

| 字段 | 内容 |
|------|------|
| 单词 | corroborate |
| 释义 | 证实，支持（证据） |
| 音标 | /kəˈrɑːbəreɪt/ |
| 例证 | The witness testimony **corroborates** the physical evidence. |
| 词根 | cor（共同）+ robor（力量）+ ate（动词后缀）→ 增强证据 |
| 拆分 | cor-rob-or-ate |

## unbelievable

| 字段 | 内容 |
|------|------|
| 单词 | unbelievable |
| 释义 | 难以置信的 |
| 音标 | /ˌʌnbɪˈliːvəbl/ |
| 例证 | The special effects are **unbelievable**. |
| 词根 | un（不）+ believe（相信）+ able（能…的） |
| 拆分 | un-be-liev-able |

## 词根词缀拆分说明

- 能拆出**前缀/词根/后缀**的，优先按词根词缀拆（如 un-believe-able）
- 词根不明显但音节清晰的，按**音节拆**（如 ar-bi-tra-ry）
- 短词（≤6 字母）按 **2 字母一组**拆，长词按 **3 字母一组**拆
